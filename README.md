# Bravo & Beyond Admin System

Separate admin platform for `admin.bravoandbeyond.com`.

Version 1 active scope:

- Authentication
- Dashboard
- CMS module
- Media library
- Contact inquiries

Reserved future scope:

- Campaign monitoring
- Client portal
- Payments
- Calendar
- Tasks
- Approvals
- Reports
- CRM

See `PROJECT_REQUIREMENTS.md` for the full working agreement.
